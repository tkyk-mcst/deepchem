概要

このアプリでできること

新薬開発や化学研究において最も重要な指標「ADMET」（吸収・分布・代謝・排泄・毒性）を、 化合物の構造式（SMILES）から即座に計算・可視化するツールです。

機能一覧

5 つの主要機能

左サイドバーのアイコンから各機能に即座にアクセスできます。

🔬

単一分子予測（Predict）

SMILES を入力して「Predict」ボタンを押すだけ。 分子の 2D 構造画像、ADMET レーダーチャート、 Lipinski の法則判定、50+ 以上の物性値を一括表示。

メイン機能
📊

バッチ予測（Batch）

複数の化合物を一度に予測。SMILES を改行区切りで 貼り付けるか CSV をアップロード。 結果はサムネイル付き表形式で表示・並び替え可能。

最大 100 分子
⚖️

分子比較（Compare）

2 つの分子を並べて比較。 Tanimoto 類似度スコア（ECFP4 フィンガープリント）、 各物性値の差分テーブル、 両方の ADMET レーダーチャートを同時表示。

Tanimoto 類似度
🔍

PubChem 検索（Search）

化合物の一般名（例：aspirin、caffeine）で検索すると SMILES・分子式・分子量を PubChem から自動取得。 「Predict Properties」で即座に予測へ。

PubChem 連携
🏠

サンプル分子（Home）

アスピリン・カフェイン・ペニシリン・ドーパミンなど 12 種の代表的分子がワンクリックで予測できる ショートカット付きホーム画面。

12 種のサンプル
⚠️

構造アラート（Alerts）

PAINS フィルター・Brenk フィルターによる 薬物候補として問題のある構造モチーフを自動検出。 創薬スクリーニングの最初のふるいかけに。

PAINS / Brenk

予測項目

計算・予測される物性一覧

RDKit による理論計算値と、DeepChem 機械学習モデルによる予測値の両方を提供します。

カテゴリ 項目名 説明 手法
物性 分子量（MW） 分子の総質量 [g/mol]。Lipinski則では ≤500 RDKit 計算
物性 LogP 水/オクタノール分配係数。脂溶性の指標。≤5 が薬物適性の目安 RDKit 計算
物性 TPSA 位相極性表面積 [Å²]。膜透過性・腸管吸収の指標。≤140 が目安 RDKit 計算
物性 HBD / HBA 水素結合供与体 / 受容体数。Lipinski則では HBD≤5、HBA≤10 RDKit 計算
物性 回転可能結合数 分子の柔軟性。≤10 が経口吸収の目安 RDKit 計算
物性 環の数 / 芳香環数 環構造の複雑さ。平面性・代謝安定性に影響 RDKit 計算
物性 Fsp3 sp3 炭素の割合。立体的複雑さの指標（>0.42 が理想的） RDKit 計算
物性 分子式 元素組成（例：C₉H₈O₄） RDKit 計算
ADMET-A 水溶性（logS） 水への溶解度の対数。吸収・製剤設計に直結 DeepChem / Delaney
ADMET-D 血液脳関門透過性（BBBP） 中枢神経系への移行性。CNS 薬では重要な指標 DeepChem / MoleculeNet
ADMET-T 毒性（Tox21） 12 種の毒性エンドポイント（NR-AR, SR-MMP など）の複合スコア DeepChem / Tox21
ADMET-T 臨床毒性（ClinTox） 臨床試験で毒性が確認された化合物との類似性 DeepChem / ClinTox
ADMET-T hERG ブロック 心毒性リスク。hERG チャンネル阻害による QT 延長の予測 RDKit 推定
ADMET-T HIV 複製阻害 抗 HIV 活性の予測 DeepChem / HIV
薬物適性 Lipinski の法則 経口薬物適性の古典的スクリーニング（「Rule of Five」) RDKit 計算
薬物適性 Veber 規則 経口生物学的利用能（回転可能結合数・TPSA による補完的判定） RDKit 計算
類似性 Tanimoto 類似度 ECFP4 フィンガープリントによる 2 分子間の構造類似度（0〜1） RDKit 計算
候補生成 アナログ候補 入力分子の構造類似体を最大 6 件自動生成 RDKit 変換

REST API

バックエンド API エンドポイント

FastAPI 製の REST API。http://localhost:8282/docs で Swagger UI が利用可能。

POST
/predict
単一 SMILES → 全物性予測（分子構造・ADMET・薬物適性・アラート）
POST
/predict/batch
複数 SMILES リスト → バッチ予測（最大 100 分子）
POST
/molecule/compare
2 分子の比較予測 + Tanimoto 類似度
GET
/molecule/image?smiles=&width=&height=
SMILES → SVG 形式 2D 構造画像
GET
/molecule/variants?smiles=&n=6
構造アナログ候補の自動生成（最大 12 件）
GET
/molecule/alerts?smiles=
PAINS / Brenk 構造アラートの確認
GET
/molecule/similarity?smiles1=&smiles2=
2 分子間の Tanimoto 類似度（ECFP4）
GET
/pubchem/search?name=
化合物名 → PubChem 経由で SMILES・分子式・分子量を取得
GET
/samples
プリセット 12 分子のリスト取得

API リクエスト例（Aspirin の予測）

# curl でテスト
curl -X POST http://localhost:8282/predict \
  -H "Content-Type: application/json" \
  -d '{"smiles": "CC(=O)Oc1ccccc1C(=O)O"}'

# レスポンス例
{
  "smiles": "CC(=O)Oc1ccccc1C(=O)O",
  "molecular_weight": 180.16,
  "logP": 1.31,
  "tpsa": 63.6,
  "hbd": 1, "hba": 4,
  "drug_likeness": { "lipinski_pass": true, "score": 4 },
  "predictions": {
    "solubility_logS": -2.14,
    "bbbp": 0.42,
    "hiv_inhibition": 0.05
  }
}
技術構成

使用技術スタック

フロントエンドからバックエンドまで、モダンな技術を採用。

📱

Flutter Web

Google 製の UI フレームワーク。CanvasKit レンダラーにより デスクトップアプリ並みのリッチな UI をブラウザ上で実現。 チャート描画に fl_chart を使用。

v3.41.6
⚡

FastAPI

Python 製の高速 REST API フレームワーク。 非同期処理（asyncio）により PubChem などの外部 API 呼び出しも ノンブロッキングで処理。/docs で Swagger UI 付き。

Python 3.9+
🧬

DeepChem

分子機械学習ライブラリ。MoleculeNet ベンチマーク データセット（Delaney・BBBP・Tox21・ClinTox・HIV）で 学習した MultitaskRegressor / Classifier を使用。

MoleculeNet モデル
⚗️

RDKit

化学情報処理の標準ライブラリ。SMILES の解析・ 分子記述子計算・SVG 描画・ECFP4 フィンガープリント・ PAINS/Brenk フィルター・3D 座標生成を担当。

オープンソース
🌐

PubChem API

NCBI が提供する世界最大の化合物データベース。 化合物名から SMILES・分子式・IUPACname・MW を リアルタイムで取得。

無料・外部 API
🎭

Playwright

E2E 自動テストフレームワーク。Flutter の CanvasKit 描画に対応し、 座標ベースのクリック操作でリグレッションテストを実施。 全画面のスクリーンショットを自動生成。

Node.js v1.59.1

起動方法

ローカル環境での起動

2 つのサーバーを起動するだけで使えます。

方法 A：バッチファイルで一括起動（Windows）

# start_servers.bat をダブルクリック、または：
cd C:\Users\takay\python379\DeepChem
start_servers.bat

方法 B：手動起動

# ① モックバックエンド（ポート 8282）
cd C:\Users\takay\python379\DeepChem
python mock_backend.py

# ② Flutter Web アプリ（ポート 3000）
python -m http.server 3000 --directory mol_app/build/web

# ③ ブラウザでアクセス
http://localhost:3000        # アプリ
http://localhost:8282/docs   # API ドキュメント（Swagger）

自動テスト（Playwright）

# 全機能の E2E テスト（スクリーンショット 12 枚を生成）
cd C:\Users\takay\python379\DeepChem
node playwright_tests/test_full.js
